# Трекер КБЖУ

Веб-приложение для отслеживания потребления калорий, белков, жиров и углеводов (КБЖУ). Приложение работает полностью в браузере без необходимости сервера.

## 🚀 Особенности

- **Адаптивный дизайн** - удобно использовать на телефоне и десктопе
- **LocalStorage** - данные сохраняются локально и не теряются при перезагрузке
- **Автоматические расчеты** - калории рассчитываются по формуле: (Белки × 4) + (Жиры × 9) + (Углеводы × 4)
- **Цели КБЖУ** - фиксированные цели: Белки 145г, Жиры 65г, Углеводы 408г, Калории 2799
- **Современный UI** - дизайн соответствует макету Figma

## 📱 Функциональность

### Главный экран
- **Дневные КБЖУ** - отображение текущих значений и целей
- **Список записей** - все добавленные записи о еде
- **Кнопка добавления** - плавающая кнопка для создания новых записей

### Добавление записи
- **Ввод КБЖУ** - поля для белков, жиров и углеводов
- **Автоматический расчет** - калории рассчитываются автоматически
- **Показатель "осталось"** - сколько осталось до достижения целей
- **Описание** - текстовое описание еды
- **Время добавления** - автоматически записывается время создания

### Управление записями
- **Удаление** - возможность удалить любую запись
- **Пересчет** - автоматическое обновление дневных итогов при изменениях

## 🛠 Технологии

- **HTML5** - семантическая разметка
- **CSS3** - адаптивные стили с поддержкой темной темы
- **Vanilla JavaScript** - чистая логика без фреймворков
- **LocalStorage API** - локальное хранение данных

## 🚀 Запуск

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в любом современном браузере
3. Приложение готово к использованию!

### Для публикации на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. В настройках репозитория включите GitHub Pages
4. Выберите источник "Deploy from a branch"
5. Выберите ветку `main` и папку `/ (root)`
6. Сохраните настройки

Приложение будет доступно по адресу: `https://ваш-username.github.io/ваш-репозиторий`

## 🔧 Настройка

    FAT: 65,         // Цель по жирам (г)

};
